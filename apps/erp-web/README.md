# MetalPro ERP/MES

ERP/MES система для металлообрабатывающих предприятий. Автоматизирует полный цикл производства — от оформления заказа до отгрузки: управление заказами, расчёт стоимости, спецификации материалов, покрытия, складской учёт.

## Стек

- **Node.js** — серверная часть
- **Prisma ORM** — работа с базой данных
- **PostgreSQL** — база данных

## Модули

### Заказы и производство
- Заказы с жизненным циклом: DRAFT → QUOTATION → APPROVED → PRODUCTION → COMPLETED → DELIVERED
- Фазовое планирование (поэтапная отгрузка)
- Дерево сборок (Assembly BOM) с неограниченной вложенностью
- Технологические маршруты (Routing): лазер, гибка, сварка, покраска

### Материалы
- Справочник материалов: сталь, нержавейка, алюминий, оцинковка
- Профили: труба, уголок, швеллер, лист, полоса и др.
- Категории материалов с иерархией
- Геометрия и нормы веса на метр/кв.м
- Закупочные профили с историей цен (RUB/USD/EUR)
- Домены: конструктив, метизы, покрытие, расходники

### Калькуляция и ревизии
- Версионирование коммерческих предложений (QuoteRevision)
- Статьи затрат: материал, резка, сварка, гибка, покраска, доставка, накладные, маржа
- Снепшоты цен на момент расчёта — история не теряется при изменении прайсов
- Ревизии сборок (AssemblyRevision) с заморозкой и сравнением версий

### Покрытия (ЛКМ)
- Справочник лакокрасочных материалов
- Системы покрытий (многослойные)
- Расчёт расхода по площади покрытия, ДФТ, потерям
- Снепшот стоимости покрытия в ревизии

### Склад и закупки
- Остатки материалов и расходников
- Резервирование под заказ/ревизию сборки
- Учёт остатков (remnants)

### Шаблоны изделий
- Повторяющиеся конструкции — создаются один раз, используются в заказах
- Иерархия узлов (до 3 уровней), нормы материалов

### Инфраструктура
- Полный аудит-лог всех изменений
- Outbox-паттерн для событий
- Управление файлами (PDF, DWG, DXF, STEP, чертежи)
- Сертификаты поставщиков
- Курсы валют

## Роли пользователей

| Роль | Доступ |
|------|--------|
| ADMIN | Полный доступ |
| MANAGER | Заказы, коммерческие предложения |
| ENGINEER | BOM, материалы, технологии |
| VIEWER | Только чтение |

## Запуск

```bash
npm install
cp .env.example .env
npx prisma migrate deploy
node server.js
```

> Автоматический деплой настроен через GitHub Actions.
